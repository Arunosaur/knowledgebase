# Markdown Cheatsheet

A quick reference guide for Markdown syntax supported in the knowledgebase.

## Text Formatting

| Syntax | Result |
|--------|--------|
| `**bold**` | **bold** |
| `*italic*` | *italic* |
| `~~strikethrough~~` | ~~strikethrough~~ |
| `` `code` `` | `code` |

## Headings

```
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

## Lists

### Unordered List

```
- Item one
- Item two
  - Nested item
```

### Ordered List

```
1. First item
2. Second item
3. Third item
```

### Task List

```
- [x] Completed task
- [ ] Pending task
```

## Links and Images

```
[Link text](https://example.com)
![Alt text](https://example.com/image.png)
```

## Blockquotes

```
> This is a blockquote.
> It can span multiple lines.
```

> This is how a blockquote looks when rendered.

## Code

### Inline Code

Use backticks: `` `code` ``

### Fenced Code Block

````
```language
your code here
```
````

## Tables

```
| Column 1 | Column 2 | Column 3 |
|----------|----------|----------|
| Cell 1   | Cell 2   | Cell 3   |
| Cell 4   | Cell 5   | Cell 6   |
```

## Horizontal Rule

```
---
```

---

## Escaping Characters

Use a backslash `\` to escape special characters:

```
\*not italic\*
\[not a link\]
```
